## Hi there, I'm Kavin-Charles 👋

I'm a developer and Linux enthusiast who builds performant, well-structured software. I run an Arch Linux system powered by [Hyprland](https://hyprland.org) — a dynamic tiling Wayland compositor — with a clean dark-themed workflow across the stack.

---

### My Linux Desktop Stack

| Component        | Choice                                                                 |
| ---------------- | ---------------------------------------------------------------------- |
| **OS**           | Arch Linux                                                             |
| **Compositor**   | [Hyprland](https://hyprland.org) (Wayland)                             |
| **Shell**        | Bash                                                                   |
| **Terminal**     | [Alacritty](https://alacritty.org) — GPU-accelerated terminal emulator |
| **Bar**          | [Waybar](https://github.com/Alexays/Waybar) — custom Wayland status bar |
| **Launcher**     | Walker / Rofi                                                          |
| **Notifications**| [Mako](https://github.com/emersion/mako) — lightweight notification daemon |
| **Editor**       | [Neovim](https://neovim.io) with [LazyVim](https://www.lazyvim.org) + Catppuccin Mocha |
| **File Manager** | Thunar                                                                 |
| **Browser**      | Firefox (Wayland-native)                                               |
| **System Monitor** | [btop](https://github.com/aristocratos/btop)                        |
| **GTK Theme**    | Catppuccin Mocha                                                       |
